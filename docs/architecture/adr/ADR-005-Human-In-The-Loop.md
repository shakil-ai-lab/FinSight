# ADR-005 — Human Review

## Status

Accepted

## Context

FinSight generates investment-related analytical findings.

Incorrect conclusions could mislead users.

Professional financial analysis requires analyst oversight.

## Decision

Every analysis passes through a mandatory Human Review checkpoint before report generation.

The graph pauses and persists state until approval.

## Consequences

### Benefits

- Higher trust
- Regulatory alignment
- Explainability
- Human accountability

### Trade-off

Longer workflow execution.

Accepted.
